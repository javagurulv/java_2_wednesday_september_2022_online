Для работы с базой данных из Java приложения нужно
открыть соединение с сервером базы данных (Connection).
В Java приложении можно открывать соединения с сервером
базы данных используя JDBC API. Но намного эффективнее
и удобнее для этой задачи использовать специальные библиотеки,
так называемые пулы соединений (connection pools).
Эти библиотеки умеют создавать не одно соединение с базой данных,
а сразу несколько. Именно поэтому такие библиотеки называются
пулы соединений.Так же с помощью таких библиотек удобно и просто
управлять созданными соединениями.

Подключите к своему приложению библиотеку пулов соединений
с реляционной базой данных:

compile('org.apache.commons:commons-dbcp2:2.8.0')

Эту строку надо добавить в build.gradle файл вашего проекта.

Библиотек реализующих функционал пулов соединений с реляционной
базой данных много, вы вправе выбрать и подключить к вашему
приложению любую из них. У каждой из этих библиотек есть своя
конфигурация, подключая библиотеку к своему проекту вы должны
будете самостоятельно разобраться с её настройками.
