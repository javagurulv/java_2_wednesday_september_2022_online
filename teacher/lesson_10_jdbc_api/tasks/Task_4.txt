В конфигурационном файле приложения application.properties
добавьте конфигурацию для подключения к серверу реляционной
базы данных.

Пример:

jdbc.url=jdbc:mysql://localhost:3306/java2
driverClass=com.mysql.jdbc.Driver
database.user.name=root
database.user.password=root

jdbc.url - это свойство содержит ip адрес (в примере это localhost)
где расположен сервер базы данных, номер сетевого порта на котором
сервер базы данных слушает соединения (в примере это 3306),
а так же название базы данных к которой будет происходить подключение.
Формат свойства jdbc url зависит от драйвера базы данных, который
вы используете для подключения к серверу базы данных.

driverClass - в этом свойстве указывается название класса из
подключённой к вашему приложению библиотеки JDBC драйвера, который
вы будете использовать для открытия соединений с базой данных. Название
этого класса находится из документации к используемой вами библиотеки.

database.user.name=root
database.user.password=root

Это свойства для указания логина и пароля пользователя от имени которого
ваше приложение будет подключаться к серверу базы данных. Это пользователь
существующий именно в рамках сервера баз данных к которому вы планируете
подключаться, не путайте этого пользователя с пользователем операционной
системы. Системы управления базами данных обеспечивают безопасность
доступа к данным. Для каждого пользователя или приложения администратор
баз данных создаёт отдельного пользователя в системе управления базами
данных и назначает ему определённые права (что может, а что не может
делать данный пользователь, например просматривать данные или иметь
права на изменения данных в базе данных). При указании имени пользователя
и пароля используйте свои значения актуальные для вашего сервера базы
данных.
