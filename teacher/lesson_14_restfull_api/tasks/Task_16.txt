Документация REST API.

Важной частью создания REST API является документация.
Необходимо чётко описать как вызывать ваш REST API,
какие параметры передавать и какой результат ожидать.

Для создания REST документации для Spring приложения
можно воспользоваться описанными в следующей статье технологиями:

https://www.baeldung.com/spring-rest-openapi-documentation

Создайте документацию для вашего REST API!
