Запустите приложение. Откройте программу Postman.

Сформируйте HTTP GET запрос на /book/{id}, вместо {id} подставив
id существующей в базе данных книги. Убедитесь, что при формировании
правильного запроса в ответ вы получаете данные книги:

{
    "errors": null,
    "book": {
        "id": 1227,
        "title": "Title",
        "author": "Author1"
    }
}

а при указании не существующего в базе данных id книги вы получаете:

{
    "errors": [
        {
            "field": "id",
            "message": "Not found!"
        }
    ],
    "book": null
}
