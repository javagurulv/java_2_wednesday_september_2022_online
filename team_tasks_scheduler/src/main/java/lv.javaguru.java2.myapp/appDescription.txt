Проект - "Tasks Scheduler"

Краткое описание:
Данное приложение выполняет основные функции планировщика задач. Планировщик задач состоит из журнала задач,
пользовательского интерфейса для добавления и удаления задачи, а также системы оповещения пользователя о событии.

Основная функциональность:
- Регистрация пользователя в системе;
- Удаление пользователя из системы;
- Вход пользователя в систему (аутентификация);
- Создание персональной задачи (одноразовой/регулярной);
- Удаление персональной задачи (одноразовой/регулярной);
- Создание групповой задачи для нескольких пользователей (одноразовой/регулярной);
- Удаление групповой задачи для нескольких пользователей (одноразовой/регулярной);
- Просмотр задач пользователем;
- Оповещение пользователя о включении его в выполнение групповой задачи (email/SMS/всплывающее окно программы);
- Оповещение о предстоящем событии (email/SMS/всплывающее окно программы).
