Проект - "Tasks Scheduler"

Краткое описание:
Данное приложение выполняет основные функции планировщика задач. Планировщик задач состоит из журнала задач,
пользовательского интерфейса для добавления и удаления задачи, а также системы оповещения пользователя о событии.

Основная функциональность:
- Регистрация пользователя в системе;
- Удаление пользователя из системы;
- Вход пользователя в систему (аутентификация);
- Создание задачи (одноразовой/регулярной);
- Изменение задачи (одноразовой/регулярной);
- Удаление задачи (одноразовой/регулярной);
- Просмотр задач пользователем;
- Оповещение о предстоящем событии.
