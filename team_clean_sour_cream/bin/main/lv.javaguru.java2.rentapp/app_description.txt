Название проекта: "Vehicle Rent".

Программа разрабатывается для обеспечения функции по аренде транспортных средств (т/с).

Функционал программы предназначен как для арендодателя:
(краткое описание функций)
- учёт т/с;
- отслеживание состояния т/с;
- учёт по сделкам (сдача в аренду/приём т/с);
- отображение информации доступности т/с;

так и для арендатора:
(краткое описание функций)
- отображение информации об услуге (услугах);
- отображение информации доступности т/с;
- возможность сделать заказ на аренду;
- получение подтверждения об резервации;
